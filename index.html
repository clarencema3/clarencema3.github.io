Projects

Here's some of the things I have worked on.

themeNB

themeNB is a clone of the popular homestay marketplace Airbnb.com and is implemented an Express and Sequelize backend to construct the database. Designed in the frontend through React/Redux to allow for an intuitive user experience through the use of wireframing.

Petsy

Petsy is a clone of the popular e-commerce website Etsy.com and is made with the collaboration of a team to design the backend with Flask and SQLAlchemy, and the frontend with React/Redux. Designed with intuitive usage across the app with the efforts of the team that I worked with on the project.

AllCourses

AllCourses is inspired by Alltrails.com and allows users to share information about golf courses. Implemented the backend with Flask and SQLAlchemy database models to allow connections between tables and reduce the number of queries required for users to be able to have a smooth experience when navigating through the app. Utilized the Google Maps API to show locations of golf courses on the app and implemented geolocating for seamless functionality of the Google Map component on the React frontend.
